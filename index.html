Slide 2

This presentation is aimed to go through the ideas behind my simple shiny app. I use this simple shiny app to show the summary stats of three small datasets, sleep, mtcars and iris. All of them can be loaded through data library. Users can view each dataset's summary stats by clicking their name at the dropdown box.

Slide 3:

  • Here are some sample code in server.R:
  datasetInput <- reactive(function() {
    switch(input$dataset,
           "sleep" = sleep,
           "mtcars" = mtcars,
       "iris" = iris
          )
  })

Slide 4:

  • Here are some sample code from ui.R:
sidebarPanel(
    textInput("caption", "Caption:", "Data Summary"),

    selectInput("dataset", "Choose a dataset:", 
                choices = c("sleep", "mtcars", "iris")),

    numericInput("obs", "Number of observations to view:", 10)
  )
